สวัสดีครับ นี่คือ หน้าหลักของ Express.js ในเว็บของเรา

Express.js เป็น Framework ที่ทำให้การพัฒนาแอปพลิเคชั่นบน NodeJS ง่ายขึ้นมาก ทั้งการสร้างเว็บแอปพลิเคชั่นและ APIs
และยังช่วยจัดการ routes, requests และ responses มี Middlewares ให้เลือกหลายตัว มีความยืดหยุ่น มีความสามารถในการขยายขนาดและน้ำหนักเบา

ตัวอย่างการใช้งานของ Express.js มีดังนี้

- สามารถเขียน Routes ซึ่งเป็น HTTP Methods ได้ใน URL ที่แตกต่างกัน
- เป็น Middlewares ซึ่งทำหน้าที่เข้าถึง Object ของ Request, Response และ next function ของ clients ที่ส่งข้อมูลมาหาเราได้
- เป็น View engine ที่ใช้ Template Engines หรือ Rendering Data เช่น static template file บนหน้า HTML ซึ่งทำให้ง่ายต่อการดูแลรักษา

ซึ่งเราจะเรียนรู้วิธีการใช้งานกันในบทต่อไป
